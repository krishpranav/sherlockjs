# sherlockjs
A remake of Sherlock made in javascript

[![forthebadge](https://forthebadge.com/images/badges/made-with-javascript.svg)](https://forthebadge.com)

# Installation
```
git clone https://github.com/krishpranav/sherlockjs
cd sherlockjs
npm install
npm start
```
